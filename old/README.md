# 🌈 Christel Detour Portfolio 🌈

### 🚀 Overview
This project presents Christel Detour, a childminder offering her services in Lyon, highlighting her activities, animations, and contact information.

### 🛒 Features
- **Presentation of Activities**: Showcasing the various activities offered for children, such as educational games and creative arts.
- **Presentation of Animations**: Highlighting special events and themed activities organized for children.
- **Contact**: Providing an easy way for parents to reach out for inquiries or to schedule visits.

### 🔧 Technologies
- **PHP**: A backend framework facilitating routing and application logic.
- **AltoRouter**: For managing application routes effectively.
- **HTML**: Structuring the content of the portfolio.
- **CSS**: Enhancing the visual design and layout of the site.
